Announcement - Integrating the TransformAble services into web applications

Integrating the Transformable Services


TransformAble has been developed so that it can be integrated into any java based web application. Some customization work, such as the selection of parts of TransformAble and the creation of a user interface that matches the web application, is required to fit TransformAble to the project specifics of the web application. For example, PreferAble and parts of StyleAble - namely the ability to use personal preferences for text and colours - have been integrated into the Sakai project.

The integration of TransformAble differs with each web application . If you would like to integrate TransformAble into your web application, please contact us for more information:
